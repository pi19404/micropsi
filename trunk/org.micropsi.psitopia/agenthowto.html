How to send your own agents



main page

You can send your own agents to Psitopia. Any agent that runs within the MicroPsi framework and uses either the SteamVehicle, Turtle or Omnidrive model can be sent to Psitopia.

Currently, you'll have to do some configuration work to achieve this and we can't guide you through all the possibilities. There is, however, a standard procedure to get a SteamVehicle agent with a Node net brain (a simpleagent, for instance) to visit Psitopia:
  • Download and install the MicroPsi toolkit (see the webdoc )
  • Start and go to Window - Preferences - Micropsi - Runtime system
  • Select a different runtime config. In the org.micropsi.core/config directory, select psitopia-client.xml
  • Enter "psitopia" in the "Server to use" field
  • Enter a user name you like
  • Uncheck "Make Server available via HTTP"
  • Restart Eclipse. Now use the toolkit as you would do it with a local system. Your agent now lives on Psitopia!
